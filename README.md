# XML_JSON_Converter_JavaFX
Ce projet est une application JavaFX qui permet de convertir des fichiers XML en JSON et des fichiers JSON en XML. L’utilisateur peut choisir entre conversion automatique via API (bibliothèque org.json) ou conversion manuelle (méthodes personnalisées pour traitement récursif des structures complexes).
